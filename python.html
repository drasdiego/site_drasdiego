O que é Python?

Python é uma linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada em diversas áreas, como desenvolvimento web, ciência de dados, automação e inteligência artificial. Criada por Guido van Rossum no final de 1989, Python ganhou popularidade rapidamente devido à sua simplicidade e legibilidade. O nome da linguagem é uma homenagem ao programa de televisão britânico Monty Python’s Flying Circus. A imagem à esquerda representa o símbolo de Python.

Olá, mundo!

Mostrar um "Olá, mundo!" na tela é uma tradição para quem está iniciando na programação. Repare que a mensagem foi inserida entre parênteses após o comando print. Esse comando, chamado de função, é o que permite exibir informações na tela. Funções são blocos de código que executam uma ação e print é uma das mais utilizadas em Python.
